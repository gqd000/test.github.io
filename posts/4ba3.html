python学习笔记（一）

这里我将记录我学习python基础的笔记来记录我的学习

一、输入与输出

1.输入与输出

a = input()
print(a)

2.评估函数和强制转换

eval() 评估函数：字符串转数字等

# 强制转换
birth = input ("Birth year: ")
a = 2020 - int(birth)
print("You age: " + str(a))
print(f"You age: {str(a)}")

强制转换实例

weigth = input("Please write you weight(g): ")
kg = int(weigth) / 1000
print("That is your weight(kg): " + str(kg))
print(kg)

3.输入提示符和字符串的拼接

first_name = input ("what is you first_name? ")
print("Hi " + first_name + "!") #字符串连接
clour = input ("what is your favourite clour? ")
print(first_name + " likes " + clour)

4.多行字符串

email = ('''
Hi,josh:

This is a letter for you!

Thank you!
The Team
''')
print (email) # 输出 ''' '''的内容（原格式）

5.格式化输出

first = "John"
last  = "Smith"
massage = first + " [" + last + "] is a coder!"#简单句子
msg = f"{first} [{last}] is a coder!"#优化：留空
print(massage)
print(msg)

二、运算

1.运算一

# 浮点数有尾数：0.1 + 0.2 ！= 0.3
# 四舍五入：round(x,d) : 对x四舍五入，截取d位小数
# 浮点数：4.3e-3  表示0.0043   9.6e5  表示960000.0
# 故 round（0.1 + 0.2，1） = 0.3
# 复数
# z = 1.23e-4+5.6e+89j  表示复数
# 获得实部：z.real
# 获得虚部：z.imag

2.运算二

print(10 * 3)#乘法
print(10 ** 3)#幂：3次方
print(10 ** 0.5) # 开平方 根号10
print(10 / 3)#除法（带小数）
print(10 // 3)#除法（）不带小数
print(10 % 3)#模运算

3.运算三

x = 10
x = x + 3
x+= 3
x-= 3
x = 10 + 3 * 2 #（=16）优先原则(同C++)
print(x)

注：
print(“转换后{:.2f}F”.format(C)) （{:.2f} 小数点保留2位小数）
print(f”{a:.2f}”) # 也是保留2位小数

三、一些数学函数

x = 2.9
print(round(x))#四舍五入
a = -985
print(abs(a))#取绝对值
print(divmod(x,y)) # 商余操作 如：divmod(10,3) = (3,1)
print(pow(x,y[,z])) # 表示 x**y % z
import math #引入math函数
print(math.ceil(2.4)) #向上取整
print(math.floor(2.9)) #向下取整
max(),min()函数
int(x)
float(x) 转换
